# AutoParts CRM - Этапы 1-11

## Что внутри

- Этап 1: главная страница, тёмный дизайн, категории, форма заявки.
- Этап 2: CRM `/admin`, Supabase, список клиентов, поиск, статусы.
- Этап 3: карточка клиента `/admin/leads/[id]`, заметки, история обращений.
- Этап 4: воронка продаж, сумма сделки, приоритет, следующий контакт, аналитика.
- Этап 5: защита админки паролем, login, logout, защита CRM API.
- Этап 6: мини-база предложений по клиенту.
- Этап 7: коммерческое предложение для WhatsApp, копирование текста, печать/PDF.
- Этап 8: дневные отчёты продавца.
- Этап 9: настройки магазина, контакты, логотип, тексты сайта, реквизиты, CSV-экспорт.
- Этап 10: внутренний каталог товаров для продавца в бутике.
- Этап 11: публичный каталог направлений + витрина популярных товаров на главной странице.

## Запуск на ноутбуке

```bash
npm install
cp .env.example .env.local
npm run dev
```

Открыть:

```txt
http://localhost:3000
http://localhost:3000/admin
```

## Supabase

Выполнить SQL:

```txt
supabase/schema.sql
```

## Админка

```env
ADMIN_PASSWORD=your-strong-password
ADMIN_SESSION_SECRET=your-long-random-secret
```

## Этап 11

На главной странице добавлены лёгкие публичные блоки:

- марки автомобилей: Toyota, Lexus, Hyundai, Kia и другие;
- направления запчастей: двигатель, ходовая, тормоза, электрика, кузов, оптика, фильтры, масла;
- топ-направления магазина;
- витрина популярных товаров;
- кнопки WhatsApp по каждому популярному товару.

Это не тяжёлый интернет-магазин. Это витрина и понятная специализация для клиента.
